## Introduction
This is the frontend of the DRDO Career Portal. It is built using React and styled with Tailwind CSS, focusing on simplicity and responsiveness.

---

## Tech Stack
- **Framework**: React.js
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Build Tool**: Vite 

---

## Features
- Responsive and modern UI.
- Login and Sign-Up functionality.
- Dynamic routing using React Router.
- Modular and reusable components.
- Tailwind CSS for styling.
